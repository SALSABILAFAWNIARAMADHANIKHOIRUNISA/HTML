Nama Tag Deskripsi
<!DOCTYPE> menentukan tipe dokumen
<html> membuat sebuah dokumen HTML
<title> membuat judul sebuah halaman
<body> membuat tubuh dari sebuah halaman
<h1> to <h6> membuat heading
<p> membuat paragraf
<br> memasukkan satu baris putus
<hr> membuat perubahan dasar kata didalam isi
<!--...--> membuat komentar
<abbr> membuat sebuah singkatan
<address> membuat kontak alamat
<b> membuat huruf bercetak tebal
<bdi> mengisolasi bagian dari teks yang dapat diformat dalam arah yang berbeda dari teks lain di luarnya
<bdo> mengganti arah teks
<blockquote> membuat sebuah bagian text yang dikutip dari sumber lain
<cite> membuat judul karya
<code> membuat potongan kode komputer di antara text
<del> membuat text yang telah dihapus dari dokumen
<dfn> membuat sebuah istilah definisi
<i> membuat sebuah bagian teks yang disesuaikan dengan mood
<ins> membuat teks yang telah dimasukkan ke dalam dokumen
<kbd> membuat input keyboard
<meter> membuat pengukuran skalar
<pre> membuat teks terformat
<q> membuat kutipan pendek
<rt> membuat sebuah anotasi/pengucapan karakter (untuk tigrafi Asia Timur)
<rp> membuat apa yang harus ditampilkan di browser yang tidak mendukung penjelasan ruby
<ruby> membuat sebuah anotasi ruby (untuk tipografi Asia Timur)
<s> membuat teks yang tidak lagi benar
<samp> membuat contoh keluaran dari program komputer
<small> membuat teks kecil
<strong> membuat teks penting
<sub> membuat teks subskrip ( seperti dalam penulisan zat kimia)
<sup> membuat teks superscripted ( seperti dalam penulisan akar kuadrat)
<time> membuat tanggal/waktu
<u> membuat teks yang memiliki gaya yang berbeda dari teks biasa lainnya
<var> membuat sebuah variabel
<wbr> membuat kemungkinan garis putus
<form> membuat sebuah form HTML untuk input pengguna
<input> membuat sebuah kontrol input
<textarea> membuat sebuah kontrol input multibaris(text area)
<button> membuat sebuah tombol yang dapat diklik
<select> membuat sebuah daftar drop-down
<optgroup> membuat sebuah kelompok pilihan yang terkait dalam daftar drop-down
<option> membuat pilihan dalam daftar drop-down
<label> membuat sebuah label untuk sebuah elemen input
<fieldset> grup unsur terkait dalam bentuk
<legend> membuat sebuah caption untuk sebuah elemen fieldset, figure, atau details
<datalist> menentukan daftar pilihan yang telah ditetapkan untuk kontrol input
<keygen> membuat key-pair generator kolom input
<output> membuat hasil perhitungan
<iframe> membuat sebuah bingkai
<img> membuat gambar
<map> membuat gambar peta
<area> membuat area dalam gambar peta
<canvas> menggambar grafik melalui scripting (JavaScript)
<figcaption> membuat sebuah caption untuk elemen figure
<figure> menentukan konten mandiri
<audio> membuat isi suara
<source> membuat sumber beberapa media untuk elemen media
<track> membuat trek teks untuk elemen media
<video> membuat sebuah video atau film
<a> membuat hyperlink
<link> membuat hubungan antara dokumen dan suber daya eksternal
<nav> membuat navigasi link
